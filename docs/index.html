Hardware-accelerated Video Transcoding

Gasket is the open-source solution for cloud-native video transcoding. Harness the power of hardware acceleration to process video faster and more efficiently.

Learn More

Failure tolerant architecture

Gasket only pushes configuration to each worker, meaning only the streams running on a worker are affected by a failure.

React based UI

Gasket comes with a React based UI that allows you to easily manage your transcoded streams.

Device Support

Gasket currently supports Nvidia GPUs, Apple VideoToolbox, AMD Alveo U30, and CPU via x264, x265.
Its underlying use of FFmpeg allows for easy integration of new devices.
